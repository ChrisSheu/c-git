## Getting Started

git clone https://github.com/ChrisSheu/c-git.git<br>
cd c-git/<br>
git checkout origin/ntp_client -b [yourbranch_name]

## Usage
```
   make ntpclient
   ./ntpclient -n [ip1].[ip2].[ip3].[ip4]
   ./ntpclient -h             //check option
```
